## Getting Started

1. **Clone the repository**  

```bash
git clone <repo-url>
cd react-app
```

2. **Install dependencies**  

```bash
npm install
```

3. **Start the development server**  

```bash
npm run dev
```

- The app will run at [http://localhost:5173](http://localhost:5173).
